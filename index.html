O Que são Hyperlinks?

Hyperlinks, normalmente chamados de links, eles são um conceito básico da web. Para explicar o que é um link, precisamos voltar aos fundamentos da arquitetura da web. Antes do advento da Web, era muito difícil acessar documentos e passar de um documento para outro. Por serem legíveis por humanos, os URLs tornam as coisas mais fáceis, mas é difícil inserir URLs longos toda vez que você deseja acessar um documento. É aqui que os hiperlinks mudam tudo completamente. O link pode associar qualquer string de texto ao URL, para que o usuário possa acessar imediatamente o documento de destino ativando o link. Os links são a inovação que torna a web tão útil e bem-sucedida.Os links se destacam do texto ao redor por estarem sublinhados e em azul. Toque ou clique no link para ativá-lo ou, se estiver usando um teclado, pressione a tecla Tab até que o link seja o foco e, em seguida, pressione Enter ou a barra de espaço.

Tipos de Links

Link Interno
Um link que as páginas pertencem ao mesmo site, sendo entre duas páginas da web, chamado de link interno. Sem a utilização de links internos, não existiria nenhum site a não ser que seja o site de uma página.
Link Exeterno
Links das suas páginas da web para as páginas da web de outras pessoas. Sem links externos, não há rede, porque a rede é a rede de páginas da web. Use links externos para fornecer informações além do que está disponível na página.
Links de Entrada
É o link de uma página de qualquer outra pessoa para o seu site, assim sendo o oposto de um link externo. Você não precisa vincular de volta quando alguém se vincula ao seu site.

Marcação de Hiperlinks

A marcação de Hiperlinks é ultilizado No HTML, os links são definidos pela tag "a".

Hiperlinks Externos ou Remotos

O destino deles são outras páginas do seu site, os links externos são aqueles que apontam para endereços diferentes.

Hiperlinks Locais

São links que direcionam à outras páginas dentro de um mesmo site.

Hiperlinks em Imagens

O destino é normalmente outra página da Web, mas também pode ser uma imagem, um endereço de email ou um programa.

Hiperlinks para Download de Arquivos

São links que apontam para arquivos e possibilitam realizar download deste arquivo.

Hiperlinks para Endereços Eletrônicos

Qualquer elemento de um hiper texto que faz referência a outro texto ou a outra parte deste texto.
